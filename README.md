
## Project Overview

The **Railway Management System API**


Admin users have additional privileges to:

- Add new trains.
- Update train details.
- Manage seat inventories.


Before setting up the project, ensure you have the following installed:

- **Python 3.10.2**
- **Django 4.0.3**
- **MySQL 8.0.28**
- **Git**
